### 五大策略详解

#### 策略① 放量上涨
- **指标**: 换手率 + 成交额 + 涨幅 + MACD金叉 + KDJ金叉
- **排序**: 按换手率降序
- **筛选**: 排除688、低流动性
- **目标**: 温和放量的上升趋势

#### 策略② 成交额排名
- **指标**: 成交量 + 封单金额 + 大单净买入
- **排序**: 按成交额降序
- **筛选**: 排除688、低流动性
- **目标**: 主力资金关注的标的

#### 策略③ 多因子量化
- **指标**: PE + PB + ROE + 净利润增长率
- **筛选**: 涨幅≥3% 且 成交额≥2亿
- **排序**: 按涨幅降序
- **目标**: 基本面良好的成长股

#### 策略④ AI技术面
- **指标**: KDJ金叉 + MACD金叉 + RSI + CCI + 量价配合
- **筛选**: 涨幅>0% 且 换手率>1% 且 主力净流入>0
- **排序**: 按主力净流入降序
- **目标**: 技术面强势的标的

#### 策略⑤ 目标价+机构
- **指标**: 目标价偏离度 + 董监高净买入 + 机构持股比例 + 北向资金
- **筛选**: 主力净流入≥1亿
- **排序**: 按主力净流入降序
- **目标**: 机构看好的标的

## 💡 操作建议

### 买入信号
✓ 综合胜率≥70% 的标的，可在支撑位布局
✓ MACD + KDJ 双金叉，且成交额≥2亿
✓ 连续3天净流入，换手率2-5%（温和放量）
✓ 涨幅≤5%，20天内有涨停历史

### 卖出信号
✗ **止损原则：股票-3% 无条件出场**
✗ 涨幅≥9.5% 且换手率≥5%，高位获利了结
✗ MACD死叉或KDJ高位钝化，减仓信号
✗ 主力净流出转向，警惕出货

### 风险管理
⚙️ 单笔仓位不超过总资金的5%
⚙️ 排除新股、低流动性标的（日均成交<1亿）
⚙️ 避免追高，等待回调后布局
⚙️ 关注政策风险、财报预期不确定性
