# EVE - A股动量策略引擎

专业的A股量化选股系统，整合五大策略，自动生成胜率评估报告。

## 项目结构

```
EVE/
├── core/               # 核心引擎
│   ├── data_source.py  # 数据层（东方财富API）
│   ├── scanner.py      # 扫描引擎（多策略整合）
│   └── report.py       # 报告生成器
├── config/             # 配置中心
│   └── settings.yaml   # 全局配置
├── utils/              # 工具模块
│   ├── logger.py       # 日志系统
│   └── helpers.py      # 辅助函数
├── strategies/         # 策略库
│   └── base.py         # 策略基类
├── tests/              # 单元测试
├── data/               # 数据存储
├── output/             # 报告输出
├── logs/               # 日志文件
├── eve_main.py         # 主入口
└── README.md           # 文档
```

## 核心特性

- **模块化设计**: 数据层、策略层、执行层分离
- **多策略整合**: 放量上涨、成交额排名、多因子量化、AI技术面、机构追踪
- **类型安全**: 完整的类型提示（Python 3.9+）
- **可配置**: YAML配置 + 环境变量覆盖
- **日志系统**: 统一日志管理
- **单元测试**: pytest测试框架

## 快速开始

```bash
# 安装依赖
pip install pyyaml pytest

# 执行扫描
python eve_main.py --scan

# 生成报告
python eve_main.py --report --output output/report.html

# 调试模式
python eve_main.py --scan --debug
```

## 五大策略

| 策略 | 排序指标 | 筛选条件 |
|------|----------|----------|
| 放量上涨 | 换手率 | 换手率≥1%，成交额≥2000万 |
| 成交额排名 | 成交额 | 成交额排序 |
| 多因子量化 | 涨幅 | 涨幅≥3%，成交额≥2亿 |
| AI技术面 | 主力净流入 | 净流入>0，换手率>1% |
| 机构追踪 | 主力净流入 | 净流入≥1亿 |

## 胜率计算

```
胜率 = 50 + 策略命中×10 + 主力净流入加分 + 换手率加分 + 涨幅区间加分
```

- 主力净流入≥5亿: +15分
- 主力净流入≥2亿: +10分
- 换手率3-8%: +10分（温和放量）
- 涨幅0-5%: +5分（合理区间）
